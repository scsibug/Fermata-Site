Fermata

Fermata is an email testing application that is simple to setup and use by anyone. It acts as a mail server, but instead of delivering mail to clients, it stores them for viewing in a web interface. This allows quick testing of software, and user acceptance and exploratory testing by business users.

What Fermata does

  • Combines a web application server and SMTP mail server, with zero configuration to get started.
  • Archives every mail message received, including attachments.
  • Presents a rich interface for finding messages by date or recipient.
  • Indexes everything for full-text search.
  • Publishes RSS feeds for all messages and each recipient.

Quick Start

  • Download and extract the latest version, double-click to start Fermata.
  • Configure the application you're testing to send mail to the machine running Fermata, on port 2500.
  • All mail activity should now be visible on http://localhost:8080
